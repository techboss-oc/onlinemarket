# Test Accounts

These accounts are set up for you to test the different roles in the application.

## 1. Seller Account

- **Role:** Seller
- **Email:** `seller@test.com`
- **Password:** `password123`
- **Features to Test:**
  - Login and view Seller Dashboard.
  - Post a new ad.
  - View "My Ads".
  - Chat with buyers (reply to messages).

## 2. Buyer Account

- **Role:** Buyer
- **Email:** `buyer@test.com`
- **Password:** `password123`
- **Features to Test:**
  - Login and view Buyer Dashboard.
  - Search for ads (e.g., "Toyota", "iPhone").
  - View single ad details.
  - **Click "Chat with Seller"** on an ad to start a conversation.
  - View and reply to messages in the Messages page.

## 3. Admin Account

- **Role:** Admin
- **Email:** `admin@test.com`
- **Password:** `admin123`
- **Features to Test:**
  - View Admin Dashboard (stats).
  - Manage Users (verify/unverify, change roles).
  - Manage Ads (approve/reject).
  - Configure Promotions.
